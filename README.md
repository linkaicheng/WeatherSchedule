# WeatherSchedule
手机天气日程通
1、	随时在有网络的地方通过应用Android操作系统的手机查询天气情况；
2、	可更换不同城市，并获取相应城市的天气信息；
3、	可更改自己的出行计划，并添加在日历备忘中；
4、	可在日历中查询、更改、添加相应的出行计划，并提醒用户； 
5、	可以把获得的天气信息通过短信分享给他人。
